# 代理服务 — ProxyCore

ProxyCore 是 MSM 支持的代理内核之一，与 Sing-Box 二选一。

::: tip 命名说明
本页中的“ProxyCore”是对外展示名称；如果命令、API 或目录中仍出现 `mihomo`，那是当前实现里的内部服务标识。
:::

在侧边栏点击 **代理服务** 展开子菜单。

## 子菜单说明

| 子菜单 | 说明 |
|--------|------|
| **概览** | 代理服务运行状态与关键指标 |
| **代理节点** | 节点列表、订阅管理、延迟测试 |
| **规则管理** | 代理分流规则与策略调整 |
| **配置编辑** | 代理配置在线编辑 |
| **连接管理** | 活跃连接查看与管理 |
| **配置管理** | 配置文件版本管理 |
| **日志查看** | 代理服务运行日志 |

## 推荐流程

1. 在初始化向导选择 **ProxyCore** 作为代理核心
2. 进入 **代理服务 > 代理节点** 导入订阅或手动配置节点
3. 在 **代理服务 > 规则管理** 调整策略规则
4. 通过 **代理服务 > 连接管理** 观察实际流量走向
5. 通过 **代理服务 > 日志查看** 排查问题

## 节点管理

- 支持订阅链接导入
- 支持手动添加单个节点
- 支持节点延迟测试
- 支持节点分组（策略组）

## 规则管理

- 域名规则、IP 规则、进程规则
- 规则提供者（Rule Provider）
- 策略组配置

## 连接管理

实时查看当前所有代理连接：

- 连接来源 IP 与目标地址
- 使用的规则和节点
- 上传/下载流量
- 支持手动关闭连接

## 注意事项

- FakeIP 网段需与 DNS 服务配置保持一致
- 修改配置后需重启服务生效

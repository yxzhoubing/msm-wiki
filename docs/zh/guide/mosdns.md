# DNS 服务

DNS 服务（基于 MosDNS）是 MSM 的核心组件，负责智能 DNS 分流、缓存与查询日志。

在侧边栏点击 **DNS服务** 展开子菜单。

## 子菜单说明

| 子菜单 | 说明 |
|--------|------|
| **概述** | 查看 DNS 服务的关键指标和详细统计 |
| **规则管理** | 管理 DNS 分流规则和黑白名单 |
| **客户端设置** | 管理客户端代理权限，支持白名单和黑名单 |
| **DNS 日志** | 查询和分析 DNS 请求日志 |
| **系统功能** | 系统功能和高级设置 |
| **配置管理** | 管理配置文件和版本 |
| **实时日志** | 查看实时运行日志和历史记录 |

## 概述

展示 DNS 服务的运行状态和关键指标：

- 服务状态、版本信息
- DNS 查询统计（总查询数、缓存命中率）
- 上游 DNS 响应统计
- 资源使用概览

## 规则管理

管理 DNS 分流规则，控制域名解析行为：

- **域名规则** — 按域名设置直连/代理/拦截
- **GeoSite 规则** — 基于 GeoSite 数据库的分类规则
- **AdGuard 规则** — 广告拦截规则管理
- **批量导入/导出** — 支持规则的批量操作

## 客户端设置

管理哪些设备使用 DNS 分流服务：

- **白名单模式** — 仅列表中的设备走代理
- **黑名单模式** — 列表中的设备不走代理
- **禁用模式** — 关闭设备管理
- **设备扫描** — 自动发现局域网设备

详见 [设备管理](/zh/guide/device-management)。

## 系统功能

高级设置与系统参数：

**🌐 全局设置**

- ECS IP：传递至上游国外DNS，以返回对此IP最友好的DNS结果，填写ipw.cn显示的ip(优先ipv6)地址或者本城市任意ipv6地址

其他功能包括：缓存管理、路由功能、转发设置、上游覆盖、功能开关等。

## 常用操作

1. **修改分流规则**
   - 进入 **DNS服务 > 规则管理**，按域名/规则类型进行调整，对某些域名强制代理或强制直连参考FAQ8
2. **设置设备名单**
   - 进入 **DNS服务 > 客户端设置**，选择白/黑名单并添加设备
3. **排查 DNS 问题**
   - 进入 **DNS服务 > DNS 日志** 查看具体请求与命中规则
   - 进入 **DNS服务 > 实时日志** 查看运行错误

## 与代理核心的协同

- FakeIP 模式需确保 **主路由** 同时配置 `28.0.0.0/8` 与 `f2b0::/18`
- 如果 MosDNS 与代理核心分体部署，DHCP DNS 指向 MosDNS，FakeIP 静态路由指向透明代理主机
- 代理核心（Clash / Sing-Box）与 DNS 服务的 FakeIP 网段、分流策略应保持一致
